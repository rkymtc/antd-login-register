## Bilgisayarınızda Çalıştırın



Projeyi klonlayın



```bash
  git clone https://github.com/rkymtc/antd-login-register
```

```bash
  cd antd-login-register

```

Gerekli paketleri yükleyin

```bash
 npm install
```

Sunucuyu çalıştırın

```bash
  npm start
```
